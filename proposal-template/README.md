# Proposal template

A fill-in LaTeX skeleton for the **project proposal** you submit to the lecturer.

1. Copy this folder into your own repository (or work here).
2. Replace every `<<< ... >>>` placeholder in `proposal.tex` with your content.
3. Build it: from the repository root run `make` → the PDF lands in `out/proposal.pdf`.
   (The build finds `style/thga-db.sty` via the root `Makefile`.)
4. Send the resulting PDF to <sboekelmann@ep1.rub.de> and wait for approval
   before you start implementing.

Target size: **2–4 pages**. See `../src/dbms_10.tex` (the assignment) for what
each section must contain, and `../example-documentation/` for the level of
detail expected in the *final* documentation.
